**Starter resume/cover‑letter build rules for Corbin Lee – please review and edit as needed.**

**Length & Formatting**
- Aim for a **one‑page** resume (max 2 if extensive project list).  
- Use **standard fonts** (e.g., Arial, Calibri, 10‑12 pt).  
- Keep margins at **0.75‑1 inch**.  
- Save/export as PDF for final submission.

**Section Order (Resume)**
1. Header (name, contact links, city/remote)  
2. Professional Summary (1‑2 concise sentences)  
3. Core Competencies / Technical Skills (bullet list)  
4. Professional Experience (reverse‑chronological, strong action verbs, quantifiable results where possible)  
5. Education (degree, institution, dates)  
6. Certifications (relevant to target roles)  
7. Optional: Projects / Open‑source contributions (if space permits)

**Tone & Language**
- **Professional, results‑focused**; start bullet points with strong action verbs (e.g., “Designed,” “Implemented,” “Optimized”).  
- Highlight impact with numbers/percentages **only when supported by data**; otherwise describe scale or complexity.  
- Avoid first‑person pronouns; keep it concise.

**ATS Friendliness**
- Use **plain text headings** (e.g., “Professional Experience,” “Education”).  
- Avoid images, tables, or unusual characters.  
- Include **keywords** from target industries: Fintech, Payments, Healthcare, Cloud, E‑commerce, Docker, CI/CD, Spring Boot, Java, Microservices, AWS, etc.  
- Use **standard date format** (Month Year – Month Year).  
- Provide location for each role (city, state) and indicate “Remote” when applicable.

**Styling**
- Use **Markdown‑compatible** heading hierarchy when drafting (e.g., `## Experience`).  
- Bullet points for responsibilities/achievements; keep each bullet ≤ 2 lines.  
- Consistent date alignment (right‑aligned or same line).  
- Bold company names and job titles for quick scanning.

**Cover Letter (optional)**
- 3‑4 short paragraphs: intro (role & referral), why you’re a fit (key achievements aligned with job), how you’ll add value, closing call‑to‑action.  
- Maintain the same professional tone and keyword emphasis as the resume.  

These guidelines provide a solid baseline; tailor the content to each specific job description.
